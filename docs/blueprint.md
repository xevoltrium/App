# **App Name**: VigourAI

## Core Features:

- AI-Powered Personalized Plan Generation: Utilize the Gemini API as a tool to generate daily workout plans customized to the user's goals (e.g., 'schlank werden'), BMI level, available equipment, and specific training preferences (e.g., back training, special workouts).
- User Authentication & Profile Management: Secure user registration and login with session persistence. Users can store and update their fitness profile details, including goals, BMI information, and equipment availability.
- Role-Based Access Control: Implement distinct user roles: 'Admin' with oversight capabilities and full modification rights for plans and users, and 'Regular User' with access to their personalized plans and profile.
- Admin Dashboard for Content & User Management: Provide administrators with an intuitive interface to review, edit, create new workout plans or methods, and manage user accounts and their associated data.
- Interactive User Onboarding & Input: A guided onboarding process where users input essential data like fitness goals, BMI stage (3 levels), available training equipment, and any desired 'special training' requests, which then informs the AI plan generation.
- Dynamic Daily Workout Plan Display: Present the AI-generated daily workout routines in a clear, actionable format, detailing exercises, specific training methods, and instructions tailored to the user's unique profile.
- Workout Completion & Basic Progress Tracking: Allow users to mark daily workouts as completed. A simple progress view showing completed days to motivate users towards their 'schlank werden' or other fitness goals.

## Style Guidelines:

- Primary color: Energetic, healthy green (#4ACC70) to represent vitality and progress, standing out cleanly against a light background.
- Background color: A serene, very light green-white (#F4FAEE) for a clean, unobtrusive canvas that promotes focus and a fresh feel.
- Accent color: A vibrant, sharp lime green (#BFF212) used sparingly for calls-to-action and key interactive elements, providing a strong visual contrast and dynamism.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, slightly tech-inspired look, complementing the AI nature of the app.
- Body text font: 'Inter' (sans-serif) for excellent readability and a neutral, professional feel across all content, especially workout instructions and user input fields.
- Use sleek, minimalist icons that clearly represent fitness activities, progress, user settings, and administrative functions, maintaining a modern and clean aesthetic.
- Employ a responsive, card-based layout with clear hierarchy to display workout plans, user profile information, and input forms. Ensure ample whitespace for a breathable and user-friendly experience.
- Subtle, fluid animations for state changes (e.g., loading plans, marking workouts complete) to provide feedback and enhance the perceived responsiveness and modernity of the interface.
